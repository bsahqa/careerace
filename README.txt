EASY EBOOK STORE

1. Open index.html to view website.
2. Edit ebooks-data.js to change ebook title, price, image, content, category, PDF link.
3. Add images in /images folder.
4. Add PDFs in /ebooks folder.
5. Replace Razorpay key in payment.js.

Important: For real sales, verify Razorpay payment on backend before delivering PDF.
