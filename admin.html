How to Edit Your Ebook Website

Only edit ebooks-data.js for ebook image, title, price, content, PDF link, category, and badge.

Example Ebook Block

{
  id: 1,
  title: "500 Selenium Interview Questions",
  price: 299,
  oldPrice: 599,
  category: "Selenium",
  image: "images/selenium.jpg",
  shortDescription: "Best ebook for Selenium interview preparation.",
  description: "Complete guide with real-time Selenium questions and answers.",
  highlights: [
    "500+ questions",
    "C# examples",
    "Framework design",
    "Freshers to experienced"
  ],
  pdfUrl: "ebooks/selenium.pdf",
  badge: "Bestseller"
}

Image Update

Put image inside images folder and update:

image: "images/selenium.jpg"

PDF Update

Put PDF inside ebooks folder and update:

pdfUrl: "ebooks/selenium.pdf"

Razorpay Key

Open payment.js and replace:

key: 'YOUR_RAZORPAY_KEY_ID'
